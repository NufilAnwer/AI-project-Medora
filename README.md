# 📌 Overview

Medora Health Assistant is an AI-based healthcare platform designed to help patients manage their health using Artificial Intelligence, Machine Learning, and Natural Language Processing.

The system provides:

- AI Medical Chatbot
- Emotional Support Companion
- Heart Disease Prediction
- Stress Prediction
- Lifestyle Recommendation
- Genetic Test Recommendation
- Patient-Doctor Communication
- Medical Record Management

# 🧠 AI & Machine Learning Modules

| Module | Technology Used |
|---|---|
| Medical Chatbot | TinyLlama + HuggingFace |
| Emotional Companion | Google Gemini |
| Heart Predictor | XGBoost |
| Stress Predictor | KNN |
| Lifestyle Advisor | TensorFlow / Keras |
| Genetic Test Predictor | XGBoost + SMOTE |
